### 🎤프로젝트 상세 소개

국가명을 검색하면 국가 기본 정보를 보여줍니다.<br />
- 출력된 국기 이미지로 배경화면 설정
- null, 영어, 숫자 등 검색어 제한
- 검색시 데이터가 없는 국가 알림창
